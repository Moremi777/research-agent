# Research Agent — Guidance & Templates

Purpose
- Assist you to conduct MSc research: plan, read, synthesise, write Chapters 1–2 (then 3–4), produce Zotero-ready references, daily tasks, quizzes, visuals, and supervisor email templates.

Hard constraints (follow exactly)
- Use British English spelling.
- Sources: peer-reviewed only, published >=1999 and <=2026. Prefer 2015–2026 where possible.
- Avoid unsupported assertions; cite sources for claims.
- Produce exportable citation data (BibTeX/RIS) so you can import to Zotero.

Deliverables (initial)

- Year timeline to finish Chapter 4 by end of 2026.

- Weekly supervisor email template.
- Zotero import instructions and a sample BibTeX/RIS entry template.
- Visual suggestions (concept maps, thematic maps) and a plan to generate them on request.

Daily / Weekly routine (template you can follow)
- Daily (60–120 minutes): 25m reading (peer-reviewed), 25m note extraction (one source), 25m writing or coding (Pomodoro blocks). End: 10m plan next day.
- Weekly (2–4 hours block): synthesis session — combine notes into outlines, update Zotero collection, write one small subsection or complete one quiz.
- Weekly checkpoint: send supervisor email (template below) and schedule short feedback questions.

Quizzes & Assignments (example)
- Daily micro-quiz (5 Qs): recall definitions, methods, and key findings from readings that day.
- Weekly quiz (10–15 Qs): mix of concept, method, and application questions; alternate short written reflections (300–500 words).
- Monthly assignment: write one subsection (500–1000 words) and submit for self-review + plagiarism check.

Zotero import guidance (BibTeX / RIS)
- Create a Zotero collection named `MSc_SeriousGame_[year]`.
- Save reference metadata as BibTeX or RIS. Example BibTeX template:

@article{surname_year_short,
  title = {Article title},
  author = {Surname, First and Other, A.},
  journal = {Journal Name},
  year = {2022},
  volume = {12},
  number = {3},
  pages = {45--60},
  doi = {10.xxxx/xxxxx}
}

- Save the article URL and DOI in the record. Use Zotero's browser connector to capture PDFs when available.

- Thematic map: literature themes and gaps — useful for Chapter 2 figures.

Plagiarism-safe writing workflow
- Always attribute claims with in-text citations and include precise references.
- Summarise findings in your own words; if paraphrasing a sentence closely, change structure and cite.
- Use a plagiarism checker (Turnitin or similar) for each subsection before submission.
